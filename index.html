Tentang Fatia

Nama saya Fatia Khoerunnisa. Saya merupakan mahasiswa baru angkatan 2025 di Universitas Mayasari Bakti. Saya mengambil jurusan teknik, prodi Informatika, dan sekarang saya sedang belajar pemrograman, khususnya di bidang web komputer menggunakan bahasa HTML dan CSS.

Untuk ke depannya, mohon bantu saya untuk belajar dan membuat saya menjadi mahasiswa yang pintar ngoding. Saya percaya dengan latihan kecil tapi konsisten, kemampuan saya dalam pemrograman akan terus berkembang.

Fun Facts 😊

  • Suka ngoding sambil dengar musik lofi.
  • Mimpi bikin website portofolio yang estetik dan interaktif.
  • Semangat belajar setiap hari walau error tetap lanjut!
